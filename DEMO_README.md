# Crash Game Demo App 💥

ایک مکمل **فوری ڈیمو** جو براہ راست چلایا جا سکتا ہے!

## کیا ہے اس میں؟ ✨

- ✅ **براہ راست شروع کریں** - کوئی سرور سیٹ اپ نہیں
- ✅ **مکمل گیم سمولیشن** - لائیو ملٹی پلائیر اپڈیٹس
- ✅ **بیٹنگ سسٹم** - شرط لگائیں اور کیشآؤٹ کریں
- ✅ **تاریخ اور اعدادوشمار** - تمام بیٹس کو ٹریک کریں
- ✅ **خوبصورت UI** - نیون گرین تھیم
- ✅ **ری سپانسیو** - ڈیسک ٹاپ اور موبائل دونوں میں کام کرتا ہے

## فوری شروعات 🚀

### 1. Install Dependencies
```bash
cd frontend
npm install
```

### 2. Update React Scripts (if needed)
```bash
npm install react-scripts@latest
```

### 3. شروع کریں
```bash
npm start
```

گیم خود بخود `http://localhost:3000` پر کھل جائے گی

## Features 🎯

### Game Tab
- لائیو ملٹی پلائیر ڈسپلے
- متعدد شرطیں بیک وقت
- فوری کیشآؤٹ
- تیز رفتار گیم لوپ

### History Tab
- تمام شرطوں کی تاریخ
- جیت/ہار کی حالت
- منافع/نقصان کا شمار

### Stats Tab
- کل شرطیں
- جیتنے کی شرح
- کل منافع/نقصان
- ملٹی پلائیر گراف
